Controls

  • To create a state:
    Double-click on an empty space on the canvas.
  • To create a transition from a highlighted state (in red):
    Shift-click on a state to point a transition towards it.
  • To highlight a state/transition:
    Single-click on a state or a transition (it would then show a red highlight).
  • To delete a state/transition:
    Press the delete key while the state or transition is highlighted.
  • To make an existing state a START state:
    Ctrl-click on it.
  • To make an existing state a FINAL state:
    Double-click on it. It would have an inner ring to it to indicate it as a FINAL state.
  • To label a highlighted state/transition:
    Type a string while the state is selected. (use '\e' for ε)
