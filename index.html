Interactive Student-Teacher FSM Game

An innovative platform designed to make learning about Finite State Machines (FSMs) both enjoyable and educational. Dive into the world of FSMs and discover their practical applications with ease.

Start Game
  • Create State: Double-click empty space
  • Create Transition From Highlighted State: Shift-click state to point at
  • Highlight State/Transition: Single-click state/transition
  • Delete State/Transition: Delete key
  • Toggle Start State: Ctrl-click state
  • Toggle Accept State: Double-click state
  • Label Highlighted State/Transition: Type (use '\e' for ε)
